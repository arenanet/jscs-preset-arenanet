[JSCS](http://jscs.info) config for ArenaNet-style code.

## Install

`> npm i jscs-preset-arenanet --save-dev`

## Usage
`> jscs -p arenanet <file>`

or you can create a `.jscsrc` file that contains

```js
{
    "preset" : "arenanet"
}
```
